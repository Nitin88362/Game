# Toffy & Fluffy preschool Kids Game Suite

A premium, interactive 3D Pixar-style suite of kids' educational games featuring **Toffy** (a cute cat in a pink dress holding a lollipop) and **Fluffy** (a superhero cat wearing a pink suit, mask, and cape).

## Games Included

### 1. 🐣 Match the Animal Babies (`one.html`)
An observation game where children match baby farm animals (calf, chick, duckling, foal, lamb) to their corresponding parents on an interactive farm.

### 2. 🍎 Fruit Adventure (`two.html`)
An interactive shopping and item matching adventure centered around colorful fruits, with rewards and outfit updates.

### 3. 🏖️ Beach Matching Adventure (`three.html`)
An advanced beach-themed matching and memory game featuring:
- **Visual & Memory Modes**: Match items face-up, or preview for 5 seconds and match from memory.
- **Progressive Levels**: Increases from 2 pairs to 10 pairs across 4 worlds (Beach Objects, Sea Animals, Colors, Shapes).
- **Interactive Connections**: SVG golden connecting lines draw dynamically between matched pairs, with confetti and starbursts.
- **Treasure Hunt**: Match cards to reveal pieces of a Treasure Map, unlocking a golden Treasure Chest with a rain of coins and a mascot victory dance.
- **Treasure Shop**: Spend pearls earned during gameplay to unlock custom Sunset and Neon beach themes and special mascot outfits.

## Technologies Used
- HTML5 Canvas & SVG animations
- Vanilla CSS3 with 3D transforms
- Vanilla JavaScript
- Web Audio API (for real-time synthesized waves sound effects, correct chimes, and incorrect buzzes)
- Speech Synthesis API (bilingual English & Hindi speech voiceovers)

## How to Play
Open `one.html`, `two.html`, or `three.html` directly in any web browser!
